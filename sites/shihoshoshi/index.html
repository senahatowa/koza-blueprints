講座の設計図 ／ 司法書士の先生へ

“登記をする人”から、“資産と会社を守る先生”へ

相続・登記・会社法務の現場知を、経営者や資産を持つ個人が学べる講座にする設計図です。

対象：相続・不動産登記・商業登記に都度対応している司法書士の先生
なぜ今これか

資格で食べてはいける。けれど、収入が“自分の時間”に縛られている。

単価を上げたくても、これ以上は働けない。自分が倒れたら全部止まる——この「時間の天井」は、もっと働いても抜けられません。抜け方が違うからです。先生の頭の中にある専門性を、一度“講座”という形にしておくと、自分が動いていない時間にも、それが働きます。

先生の専門なら、こう組める

講座カリキュラムの設計例（6モジュール）

MODULE 01

揉める相続・揉めない相続を分ける、生前の3手

誰もが他人事にしていて、起きると一番こじれるテーマ。専門家の早期介入が価値

MODULE 02

会社を継ぐ・畳む・売る、登記から見た事業承継の現実

経営者が後回しにして手遅れになる領域。先生だから語れる順序

MODULE 03

不動産を“持つ・貸す・遺す”ときの名義と権利の基本

資産家ほど知りたい、検索では危ういテーマ

MODULE 04

増える“おひとりさま”の財産と意思を守る仕組み

成年後見・家族信託・遺言。需要が伸びる成長領域

MODULE 05

会社の登記放置が招く、見えないリスク

役員変更・本店移転の放置が後で効く。中小経営者の盲点

MODULE 06

専門家への頼みどき＝手遅れにならない判断軸

受講者が“今動く理由”を理解する＝相談に繋がる設計

これが“資産”になる、ということ

顧問やスポット相談は、先生が動いた分だけ。講座は、一度作れば動いていない時間にも価値を届けます。

今自分が動く → 収入。止まれば、ゼロ。
これから知識が働く → 自分の時間が戻る。
