講座カリキュラムの設計例（6モジュール）

MODULE 01

毎回1対1で教えている“同じ流れ”を、最初の講座1本にする

再現性のあるセッションを一度収録すれば、何人にでも届く

MODULE 02

あなたのセッションの“勝ちパターン”を言語化する

属人的な感覚を型にする。ここが講座の有料の核になる

MODULE 03

受講生が“付きっきり無し”で成果を出す、ステップ設計

脱・伴走。あなたが手を離しても成果が出る構成にする

MODULE 04

1対1を全部やめずに、講座と高単価個別を両立させる導線

講座で広く取り、個別は厳選・高単価に。収入の柱を二本にする

MODULE 05

高単価を正当化する、講座ならではの価値の積み方

セッションの切り売りより、体系化された講座の方が高く売れる理由

MODULE 06

集客を“自分の発信頼み”から外す、売れ続ける仕組み

毎日投稿しないと止まる状態から、講座が勝手に売れる状態へ
