講座の設計図 ／ 税理士の先生へ

“申告を代行する人”から、“お金の不安を消す先生”へ

顧問の現場で毎日答えている「経営者のお金の悩み」を、一度きりで終わらせず、何度も売れる講座にする設計図です。

対象：決算・節税・資金繰りの相談に、毎回ゼロから個別対応している税理士の先生
なぜ今これか

資格で食べてはいける。けれど、収入が“自分の時間”に縛られている。

単価を上げたくても、これ以上は働けない。自分が倒れたら全部止まる——この「時間の天井」は、もっと働いても抜けられません。抜け方が違うからです。先生の頭の中にある専門性を、一度“講座”という形にしておくと、自分が動いていない時間にも、それが働きます。

先生の専門なら、こう組める

講座カリキュラムの設計例（6モジュール）

MODULE 01

数字が読めない経営者が、まず最初に見るべき3つの数字

顧問先に毎回口頭で説明している内容。動画1本にすれば、もう同じ説明を繰り返さなくていい

MODULE 02

税務署に睨まれない“攻めの節税”と“やってはいけない節税”の線引き

先生の頭の中にある判断基準を言語化する。これは検索しても出てこない＝有料の核になる

MODULE 03

資金繰りで会社を潰さないための、現金の見方とタイミング

決算書からは見えない“現金の動き”。経営者が一番怖がっているテーマ

MODULE 04

税理士に聞きづらい、社長の“個人のお金”の最適化

役員報酬・退職金・保険。顧問契約の外側にある、踏み込めば一番感謝される領域

MODULE 05

補助金・融資で“通る会社”が事前にやっていること

申請のテクニックでなく、日頃の数字の整え方。先生だから語れる視点

MODULE 06

顧問料を上げても解約されない、価値の伝え方

先生自身の単価にも効く。受講した経営者が“顧問の重要性”を理解する設計

これが“資産”になる、ということ

顧問やスポット相談は、先生が動いた分だけ。講座は、一度作れば動いていない時間にも価値を届けます。

今自分が動く → 収入。止まれば、ゼロ。
これから知識が働く → 自分の時間が戻る。
