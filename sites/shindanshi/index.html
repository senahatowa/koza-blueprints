講座の設計図 ／ 中小企業診断士の先生へ

“診断する人”から、“経営を伴走する先生”へ

経営改善・補助金・事業計画の知見を、経営者が繰り返し学べる講座にする設計図です。

対象：経営相談・補助金・事業計画の支援を都度こなしている診断士の先生
なぜ今これか

資格で食べてはいける。けれど、収入が“自分の時間”に縛られている。

単価を上げたくても、これ以上は働けない。自分が倒れたら全部止まる——この「時間の天井」は、もっと働いても抜けられません。抜け方が違うからです。先生の頭の中にある専門性を、一度“講座”という形にしておくと、自分が動いていない時間にも、それが働きます。

先生の専門なら、こう組める

講座カリキュラムの設計例（6モジュール）

MODULE 01

数字でなく“流れ”で会社を見る、経営の地図の描き方

社長が腹落ちする全体像。コンサルの一番おいしい入口を型にする

MODULE 02

売上を“運”でなく“設計”にする、商売の構造分解

先生が現場で何度も使っているフレームを言語化する

MODULE 03

補助金を“もらって終わり”にしない投資の組み立て方

申請支援の一歩先。継続支援に自然に繋がる

MODULE 04

人に任せて回る会社へ：社長が現場を抜ける手順

多くの一人社長の悩みそのもの。共感が深い

MODULE 05

価格を上げても客が離れない、値決めの考え方

中小の最重要テーマ。診断士の知見が最も金になる

MODULE 06

先生自身の経験談で語る、つまずきと打ち手

教科書でなく実戦。受講者が信頼する核になる

これが“資産”になる、ということ

顧問やスポット相談は、先生が動いた分だけ。講座は、一度作れば動いていない時間にも価値を届けます。

今自分が動く → 収入。止まれば、ゼロ。
これから知識が働く → 自分の時間が戻る。
